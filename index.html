Commands

Commands are a way to give the dokis actions such as hug, headpat and tickle.
You can use the command by putting the prefix and the command name.

Chat

Chat is way to interact with the dokis, saying stuff like hello and good morning, some commands are available through chat as well.
To talk, you have to mention the doki at the beginning.

Note: If the dokis don't respond, it's most probably due to the toggle command setting it off.
